# مبيعاتي v1.0.3

## إصلاح مهم

- إصلاح خطأ نافذة تثبيت التحديث في PowerShell.
- إصلاح مشكلة تحويل نص عربي إلى رقم داخل شريط التقدم.
- حفظ سكربت التحديث بترميز UTF-8 BOM حتى تظهر النصوص العربية بشكل صحيح.

## النتيجة

الآن عند تحميل التحديث:

- تظهر نافذة تثبيت واضحة.
- يتم فك الضغط.
- يتم تركيب الملفات.
- يعاد تشغيل التطبيق تلقائياً.
- تظهر ملاحظات التحديث من ملف `read.md`.
