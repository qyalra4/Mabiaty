# مبيعاتي v1.0.1

## الجديد

- إعادة ترتيب شاشة الأدوات على شكل تبويبات واضحة.
- إضافة واجهة تحديثات داخل التطبيق مع حقل رابط GitHub أو رابط ZIP مباشر.
- إضافة تحميل التحديث مع شريط تقدم وإعادة تشغيل التطبيق بعد الاستبدال.
- تحسين مركز الدعم بمحتوى عملي: أسئلة شائعة، خطوات إرسال طلب دعم، وروابط مساعدة.
- عرض نافذة ملاحظات التحديث عند فتح التطبيق بعد تثبيت إصدار جديد.

## تحسينات الواجهة

- تقسيم الأدوات إلى: قاعدة البيانات، الصيانة، التحديثات، والنشر.
- إضافة شرح داخل واجهة التحديث حتى يعرف المستخدم ماذا يحدث أثناء الفحص والتحميل.
- تحسين عرض ملاحظات الإصدار داخل التطبيق بصيغة مقروءة.

## ملاحظات مهمة

- ملف التحديث ZIP يحتوي ملفات البرنامج فقط.
- لا يحتوي ملف التحديث على قواعد بيانات المستخدم أو بيانات المتاجر.
- عند تثبيت التحديث يتم استبدال ملفات البرنامج وإعادة تشغيل التطبيق تلقائياً.
