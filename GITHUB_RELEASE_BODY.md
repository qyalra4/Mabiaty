# مبيعاتي v1.0.1

هذا الإصدار يضيف واجهة تحديثات داخل التطبيق مع تحسين شاشة الأدوات ومركز الدعم.

## طريقة التجربة

1. حمل ملف `Mabiaty_Update_v1.0.1.zip` من Assets.
2. افتح التطبيق.
3. ادخل إلى `الأدوات` ثم تبويب `التحديثات`.
4. ضع رابط المستودع بصيغة:
   `https://github.com/USER/REPO`
5. اضغط `فحص التحديث`.
6. اضغط `تحميل وتثبيت التحديث`.

## التغييرات

- شاشة أدوات مرتبة بتبويبات.
- مركز دعم بمحتوى كامل ومنظم.
- دعم GitHub Releases للتحديث التلقائي.
- نافذة تعرض ملاحظات الإصدار بعد التحديث.

## ملاحظة

ارفع ملف ZIP كـ Release Asset، وليس داخل سورس المشروع فقط.
