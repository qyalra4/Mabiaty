# مبيعاتي v1.0.2

هذا الإصدار يصلح تثبيت التحديث بعد التحميل، ويضيف نافذة تركيب واضحة، ويحسن شاشة الأدوات وشاشة حسابي.

## طريقة التجربة

1. حمل ملف `Mabiaty_Update_v1.0.2.zip` من Assets أو استخدم الرابط المباشر.
2. افتح التطبيق.
3. ادخل إلى `الأدوات` ثم تبويب `التحديثات`.
4. ضع رابط المستودع بصيغة:
   `https://github.com/qyalra4/Mabiaty`
5. أو ضع رابط ZIP مباشر:
   `https://github.com/qyalra4/Mabiaty/raw/main/Mabiaty_Update_v1.0.2.zip`
6. اضغط `فحص التحديث`.
7. اضغط `تحميل وتثبيت التحديث`.

## التغييرات

- إصلاح سكربت تثبيت التحديث.
- نافذة تقدم أثناء فك الضغط وتركيب الملفات.
- حذف تبويب النشر من شاشة الأدوات.
- تحسين ألوان التقدم والأيقونات.
- إعادة تصميم شاشة حسابي.

## ملاحظة

ارفع ملف ZIP كـ Release Asset للإنتاج النهائي. وللتجربة يمكن استخدام رابط الملف المباشر من الفرع `main`.
