# مبيعاتي v1.0.2

## الجديد في هذا التحديث

- إصلاح تثبيت التحديث بعد التحميل من GitHub.
- ظهور نافذة واضحة أثناء فك الضغط وتركيب ملفات التحديث.
- إعادة تشغيل التطبيق تلقائياً بعد اكتمال التحديث.
- عرض ملاحظات التحديث من ملف `read.md` عند أول فتح بعد تثبيت التحديث.
- تحسين شاشة الأدوات وحذف تبويب النشر.
- تحسين ألوان شريط التقدم ليطابق لون أزرار التطبيق والقائمة الجانبية.
- إصلاح ألوان الأيقونات داخل الأزرار حتى تتناسق مع لون النص.
- إعادة تصميم شاشة حسابي بالكامل بشكل احترافي ومنظم.

## ملاحظات

- هذا التحديث لا يغير قواعد بيانات المستخدم.
- هذا التحديث لا يحذف بيانات المتاجر.
- إذا كان التطبيق مثبتاً داخل Program Files فقد يحتاج صلاحيات مدير أثناء تركيب التحديث.
